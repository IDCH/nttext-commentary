Implements a system for organizing references to Biblical material, for parsing human-entered verse references and for supporting multiple Bible organization strategies.

This is based, in part on work by Weston Rueter and Robert Hunt. Please see:

  • Reference Parsing: http://code.google.com/p/open-scriptures/source/browse/branches/php-prototypes/reference-parser.lib.php
  • Bible Organization Code Repository: https://github.com/openscriptures/BibleOrgSys
  • Bible Organization Blog Entries: http://openscriptures.org/blog/2011/01/bible-organisational-system
